Typography Sample

What is this?

This page contains several common tags that you will see on almost any web page. The goal is to style all of the type here so reads well.

Get started by looking at the page without any styles. Next apply some styles. Use your judgement and get feedback from anyone available. Listen to what people think and and apply their suggestions.

Look closely at these things:

  • Font family - Choose Serif or San Serif
  • Font size - Set the size of the font to make it legible
  • Color & background color - Strive for good contrast
  • line height - Again strive for legibility

What to style? Style all these tag names. Remember, CSS styles are inherited. For styles that you want to be commonly across many elements set the style on the HTML tag: html {...}.

  • html - Set all of the base font styles:
    • font-family
    • font-size
    • line-height
    • color
  • h1-6 - Headings, set the font size using a relative unit like em
  • p - set the margin-top and margin-bottom
  • strong - Use color and font-weight to
  • em
  • cite
  • blockquote
  • figcaption
  • code

Details

Look closely at the details. Some of tags are usually used together and others are used to enhance communication

Images

The figure tag and figcaption are used with images. You can style these to make the images display better and the caption fell like it's labeling the image.

Blockquotes

The blockquote is used to show you've taken text from another source and are reproucing it here. The cite tag is used to say where it came from.

Strong and em

The strong and em tags are used to add a voice to your message. Use strong when you want to say something with a strong voice. This can be something important or when you feel like it needs to be said forcefully.

The em tag stands for emphasis. Use this when you want to say something with emphasis. Use this when you want to call attention to particular words or phrases.

You should write CSS styles for both of these tags. Use color font, weight, and font style.

em vs rem

The box below has a font-size of 20px.

Inherit this text is inherits it's font-size 20px.

font-size: 1em this text is 1em. It's font-size is 20px. This is 1 times the inherited font-size of 20px.

font-size: 1.5em this text is 1.5em. Here the font-size is 30px. That's 1.5 times the inherited font-size of 20px.

font-size: 18px this text is 18px.

font-size: 1rem this text is 1rem. Here the size is 16px because the root font-size is 16. That's 1 times the root font-size.

font-size: 1.25rem this text is 1.25rem. Here the size is 20px because the root font-size is 16. That's 1.25 times the root font-size of 16px.
